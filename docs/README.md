# MANUAL WEB POO

## ¿Qué hay en nuestro landing page?
- Interfaz que simula terminales antiguas: Animaciones de escritura, cursor parpadeante y efectos neón.
- Renderizado Dinámico: Los temas se cargan desde una base de datos en JavaScript.
- Experiencia Interactiva: Secciones de ejercicios plegables con efectos de sonido retro (Web Audio API).
- Diseño Responsivo: Adaptado para terminales móviles y de escritorio.
## Tecnologías Utilizadas
- HTML5 / CSS3: Variables CSS, Flexbox, Grid y animaciones personalizadas.
- JavaScript (ES6+): Manipulación del DOM, lógica de filtrado y AudioContext.
- Assets: Google Fonts (VT323), Lucide Icons y Giphy para el fondo dinámico.
## Cómo usarlo 
1. Clona el repositorio
2. Abre index.html en tu navegador.
## Estructura
/css: Estilos.
/js: Lógica de renderizado y datos de los temas.
index.html: Punto de entrada del sistema.
/assets: contiene gifs, sonidos, iconos y fuentes tipográficas utilizadas.
